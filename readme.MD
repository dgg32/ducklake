
  
  

# Introduction

  

This repository contains code and data for my article "[DuckLake Medical Knowledge Graph](https://dgg32.medium.com/ducklake-medical-knowledge-graph-6b9a6bd57543?postPublishedType=initial)".



# Prerequisite


MotherDuck account

Supabase and GCS account

Python

DuckDB


# Run

1. Run supabase_gcs_ducklake.ipynb to set DuckLake up on your Supabase and GCS

2. Run motherduck_ducklake.ipynb to set DuckLake up on your MotherDuck

3. Run query.ipynb to query the DuckLake
